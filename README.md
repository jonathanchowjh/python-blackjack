## Python Blackjack
Simple pygame based game for blackjack
<p align="center">
  <img src="./assets/img4.png" width="80%" title="blackjack image">
</p>

### Setup
```sh
python3 -m venv path/to/venv
source path/to/venv/bin/activate
pip3 install pygame
python3 main.py
```
